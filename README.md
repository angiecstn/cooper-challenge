# cooper-frontend
Mobile fitness tracking application (Ionic &amp; Rails)

### Prerequisites

* [Node.js] 
* [Ionic & Cordova] 


## Running the tests

* npm run test-coverage   // for unit testing
* npm run e2e  // for feature testing
* open coverage/index.html // to check test coverage 


## Built With

* [Ionic] The web framework used
* [e2e] - Feature Testing
* [npm] - unit testing
* [node.js] 
* [Karma]
* [Jasmine] 
* [Protraktor]
